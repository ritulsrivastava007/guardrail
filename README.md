# 🛡️ GuardRail

### AI-powered scam detection & financial safety system

> A security layer designed to detect suspicious financial activity before users fall victim to scams.

---

## 🤔 Why GuardRail?

Financial scams are evolving.

Attackers no longer just steal information — they manipulate people using:
- fake transaction messages
- urgency tactics
- social engineering
- remote access scams

The biggest weakness is often not the technology, but the moment of panic.

GuardRail explores a simple idea:

**What if your phone could warn you before you make a dangerous decision?**

---

## 💡 What does it do?

GuardRail simulates an AI security assistant that:

📩 Detects suspicious transaction notifications  
⬇️  
🧠 Analyzes sender reputation and transaction behavior  
⬇️  
🚨 Calculates risk level  
⬇️  
🔒 Protects the user with real-time warnings

---

## ✨ Features

### 🧠 AI Risk Analysis
Simulates fraud detection using:
- Sender verification
- Transaction pattern analysis
- Device risk checks
- Behavioral signals

### 🔐 Zero-Trust Transaction Flow
Suspicious transactions are isolated until verification is completed.

### 👴 Designed for Accessibility
A security system should be usable by everyone, including non-technical users.

### 🎭 Demo Simulation
Includes:
- Safe transaction scenario
- Scam transaction scenario
- AI analysis visualization
- Protection flow
